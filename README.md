## 🚀 Projects

### 🎬 Movie Review Sentiment Classification
> 클래스 불균형 문제를 분석하고 ensemble 전략을 통해 성능을 개선한 프로젝트

🙋‍♂️ My Role

- 1인 단독 프로젝트 (데이터 분석, 모델링, 실험 전 과정 수행)

🔬 Contribution

- 미세한 가중치 조절([0.22, 0.21, ... ])을 통한 Weighted Ensemble 적용
- TAPT(Task Adaptive Pretraining)를 활용한 도메인 적응

💡 Insight

- 단순 기법(Loss function 변경) 적용보다 문제의 구조를 이해한 앙상블/TAPT 전략이 성능에 더 큰 영향을 미침을 확인

👉 https://github.com/arnixs115/boostcamp8-review-sentiment-classification
